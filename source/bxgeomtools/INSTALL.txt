
geomtools installation instructions:
------------------------------------

1 - Third-party software setup :
--------------------------------

geomtools depends on the following libraries and/or programs :
 - CLHEP (through Cadfael)
 - CAMP (through Cadfael)
 - gnuplot (through Cadfael)
 - Root (through Cadfael)
 - mygsl (through Bayeux)
 - materials (through Bayeux)


2- Plain CMake installation :
-----------------------------

Change to the geomtools source base directory :
{{{
bash$ cd <geomtools_source_base_dir>
}}}

Create a 'build' directory :
{{{
bash$ mkdir build
bash$ cd build
}}}

Configure the installation to the '${HOME}/software' directory.
Note: you must choose a directory where you have write access.

{{{
bash$ cmake \
   -DCMAKE_INSTALL_PREFIX:PATH=${HOME}/software \
   -DCMAKE_BUILD_TYPE:STRING=Release \
   -Dmygsl_DIR=<mygsl_CMake_config_file_dir> \
   -Dmaterials_DIR=<materials_CMake_config_file_dir> \
   -DCMAKE_MODULE_PATH=<Cadfael_CMake_find_package_files_dir> \
   -DGEOMTOOLS_WITH_DEBUG=0 \
   -DGEOMTOOLS_WITH_BIO=1 \
   -DGEOMTOOLS_WITH_DOC=1 \
   -DGEOMTOOLS_WITH_TEST=1 \
   -DGEOMTOOLS_WITH_GNUPLOT_DISPLAY=1
   ..
}}}

This prints :
{{{
-- The C compiler identification is GNU
-- The CXX compiler identification is GNU

...

-- Configuring done
-- Generating done
-- Build files have been written to: <geomtools_source_base_dir>/build
}}}

Build :
{{{
bash$ make
}}}

Generate the documentation :
{{{
bash$ make doc
}}}

Perform tests :
{{{
bash$ make test
}}}

Clean test material :
{{{
bash$ make clean_test
}}}

Installation (here in the '${HOME}/software' directory) :
{{{
bash$ make install
}}}

Remove the temporary build directory :
{{{
bash$ cd build
bash$ rm -fr build
}}}


3 - Production CMake installation using the 'pkgtool' wrapper :
---------------------------------------------------------------

Change to the geomtools source base directory :
{{{
bash$ cd <geomtools_source_base_dir>
}}}

Getting help :
{{{
bash$ ./pkgtools.d/pkgtool --help
}}}

Run the following configure/build/install sequence :
{{{
bash$ ./pkgtools.d/pkgtool check
bash$ ./pkgtools.d/pkgtool configure --install-prefix ${HOME}/software
bash$ ./pkgtools.d/pkgtool build --number-of-tasks auto
bash$ ./pkgtools.d/pkgtool doc
bash$ ./pkgtools.d/pkgtool test
bash$ ./pkgtools.d/pkgtool install
}}}

De-installation :
{{{
bash$ ./pkgtools.d/pkgtool uninstall
}}}

Full reset of the package :
{{{
bash$ ./pkgtools.d/pkgtool reset
}}}


4 - Setup :
-----------

The following instructions applies to the bash shell.

Load the definition of the geomtools setup function :
{{{
bash$ source ${HOME}/software/etc/geomtools_setup.sh
NOTICE: do_datatools_setup: Please setup 'geomtools' with 'do_geomtools_setup'.
}}}

Invoke the setup function :
{{{
bash$ do_geomtools_setup
NOTICE: do_geomtools_setup: Setup geomtools...
NOTICE: do_geomtools_setup: Setup geomtools done.
}}}

Use geomtools :
{{{
bash$ which geomtools-config
/home/<login>/software/bin/geomtools-config
bash$ geomtools-config --version
0.1.0
bash$ geomtools-config --cflags
...
bash$ geomtools-config --ldflags
...
bash$ geomtools_check_setup -GRD -c ${GEOMTOOLS_TESTING_DIR}/config/test-1.0/test_manager.conf
}}}
