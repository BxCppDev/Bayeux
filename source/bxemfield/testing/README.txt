README.txt
----------

These  sample files and  are used  during the  testing step  of emfield.
They are  provided as  is as  examples of some  possible use  of emfield
classes.

----------------------------------------------------------------------

Run test scripts from the '__build-Linux-x86_64' build directory :

bash$ EMFIELD_DATA_DIR=$(pwd) \
  __build-Linux-x86_64/testing/test_emfield_manager \
  --plot --plot2 \
  -Ex10 -Ex10 \
  -Bx10 -Bx10
