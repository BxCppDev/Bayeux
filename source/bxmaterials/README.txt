
materials  is  a C++  program  library  developped  by Benoit  Guillon
<guillon@lpccaen.in2p3.fr> at  LPC Caen (CNRS/IN2P3)  and ENSICAEN. It
proposes some  classes to represent and  manipulate isotopes, chemical
elements  and materials.  It is  compatible with  the GDML  and GEANT4
approach.

It depends on the geomtools program library and thus datatools, mygsl,
CLHEP.

There are other contributors to this library :
 - François Mauger (LPC Caen (CNRS/IN2P3), Université de Caen Basse Normandie) :
   bug fixes, build system
 - Xavier Garrido (LAL (CNRS/IN2P3), Université Paris-Sud) : bug fixes
 - Mathieu Bongrand (LAL (CNRS/IN2P3), Université Paris-Sud) : improvements
 - Ben Morgan (University of Warwick) : CMake build system
