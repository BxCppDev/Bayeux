These  sample files and  are used  during the  testing step  of cuts.

classes.

--------------------------------------

From a pkgtools build :

Test the 'test_manager' executable :

  bash$ CUTS_TESTING_DIR=$(pwd)/testing __build-Linux-x86_64/testing/test_manager

Also :

  bash$ cd testing/
  bash$ ./test_manager.sh
