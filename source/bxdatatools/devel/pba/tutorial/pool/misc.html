Using an archive without header

As mentioned before, it is possible to create a portable binary archive without header. This is done by passing the boost::archive::no_header flag to the constructor of the output archive. The input archive used to read (deserialize) the data file must also (symmetrically) be given the same flag.

include(`tutorial_pba_2.cpp.html')

The resulting pba_1.data file contains the following bytes: 

include(`pba_1.bin')
where there are no leading header bytes and thus only the value of the integer (876)10=(1101101100)2 is stored using 2 bytes:

00000011 01101100
3 108
