
mygsl is a C++ library which provides some C++ wrapper classes for the
GSL library as well as related numerical tools.
mygsl depends on  the datatools (from Bayeux) and  GSL libraries (from
Cadfael).

mygsl is distributed as a module of the Bayeux software suite.
