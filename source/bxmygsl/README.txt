
Bayeux/mygsl is a C++ library  which provides some C++ wrapper classes
for the GSL library as well as other numerical tools.

Bayeux/mygsl  depends on  the Bayeux/datatools  and the  GSL libraries
(provided by Cadfaelbrew).

Bayeux/mygsl is distributed as a module of the Bayeux software suite.
