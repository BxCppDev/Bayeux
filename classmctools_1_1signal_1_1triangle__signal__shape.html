Detailed Description

Triangular signal shape.

The signal shape :

duration
:<------------>:
t0 tpeak t1
---------+------+-------+------------------> time
: \ : /
: \ Q : /
: \ : /
amplitude \ : / polarity == negative
: \ : /
: \: /
