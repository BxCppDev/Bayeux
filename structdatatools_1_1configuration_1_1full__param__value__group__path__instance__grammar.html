Detailed Description

template<typename Iterator>
struct datatools::configuration::full_param_value_group_path_instance_grammar< Iterator >

Parser for a full variant parameter value group path.

Supported format:

"name:a"
"name:a@grp1"
"name:a[12]"
"name:a[12]@grp1"
"name:a/b/c"
"name:a/b/c@.grp"
"name:a[12]/b/c"
"name:a/b/c[12]@grp1"
"name:a[12]/b/c[12]@grp1"
"name:a/b/c[12]@grp1"
"name:a[12]/b/c[12]/d/e@grp1"
