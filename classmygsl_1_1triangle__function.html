Detailed Description

General asymmetric triangle function.

^
: head tail
: width width
: :<-->+<--->: ^
: : /:` : :
: : / : ` : :
: : / : ` : : amplitude
: :/ : ` : :
---+-------+....+.....+------v-----> x
