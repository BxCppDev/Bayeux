Detailed Description

Top-level namespace of the Bayeux/dpp module library.

Standard BRIO file format for physics event data is :

Store: "GI" (general informations as 'DPP_DU::properties' [optional])

  • header (DPP_DU::properties) [optional]
  • footer (DPP_DU::properties) [optional] Store: "ER" (event record as 'DPP_DU::things' (things))
  • event record # 0
  • event record # 1
  • event record # 2
  • ...
  • event record # N
