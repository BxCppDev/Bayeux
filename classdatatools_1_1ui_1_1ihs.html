static

Check if a path has a leading scheme. If scheme is given, its specific value is checked.

Examples: "/path/to/resource" -> has no scheme "foo:/path/to/resource" -> has scheme "bar:/path/to/resource" -> has scheme "bar:/path/to/resource" -> has scheme "bar"
