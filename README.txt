
Bayeux - An aggregation of C++ library tools for scientific experiments.


This is the legacy version of  Bayeux. It covers Bayeux' release up to
tag 0.9.99. Above version number  0.9.99, a new Bayeux aggregator must
be used.


Description
-----------

datatools :  Serializable data structures based on Boost.
             [version < 5]

brio :       A Boost over ROOT I/O serialization system (extends datatools).
             [version < 4]

mygsl :      C++ wrapper and extensions to the GNU Scientific Library.
             [version < 4]

geomtools :  Generic tools for working with experiment geometries and
             provide input to simulation tools (i.e. GDML/Geant4).
             [version < 4]

cuts :       Generic tools for arbitrary data selection (a.k.a. cuts).
             [version < 4]

materials :  Description of isotopes , elements and materials and tools
	     for input to simulation tools (i.e. GDML/Geant4).
             [version < 4]

genvtx :     Vertex random generator tools for input to
             simulation tools (i.e. Geant4).
             [version < 4]

genbb_help : C++ wrapper and extensions to the Decay0/GENBB program
             by Vladimir Tretyak, for input to simulation tools (i.e. Geant4).
             [version < 5]

emfield :    Electromagnetic field modelling and management.
             [version < 1]

trackfit :   Track fitting in a NEMO-like detector.
             [version < 3]
